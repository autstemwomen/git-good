## gitGood
### Setup git / GitHub
1. Create/login to a [GitHub account][1].
2. Install [git][2].
3. Install [GitHub Desktop][3], and sign in.
4. Install [VSCode][4].
5. Done

[1]: https://github.com/
[2]: https://git-scm.com/downloads
[3]: https://github.com/apps/desktop
[4]: https://code.visualstudio.com/
